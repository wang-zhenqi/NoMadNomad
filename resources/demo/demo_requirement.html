Summary

We are looking for someone to help us build an image solver for a website for research. They ask you to solve some challenge, and then you have to take action on that challenge. Your role will NOT have to involve the action taking - we only need your help to build the right Python-based system that can solve the images at a high scale locally. We are not sure yet if LLMs will be able to solve the images (or solve them fast enough), so you will first attempt to solve it with a few LLM models. If it can reliably solve it, great! If not, you'll have to build some human-in-the-loop system which works with some image solving service, or possibly we need our own humans-in-the-loop (Mechanical Turk, freelancers, etc). We envision the project to generally look like this: 1. Download 50 or more samples of the images yourself 2. Map the images to the correct answer manually yourself 3. Then experiment with different LLM providers to see if they can answer it accurately, and on time. We have to solve the images within 5s, so it must be fast. 4. If LLMs can't do it, then we have to tie into some human-in-the-loop service where humans are given the images and they pass back the results to us. You will not + should not try building your own custom image solver - it's too big of a scope. We are only interested in using ready-made solutions (LLMs, or otherwise), or humans in the loop. We prefer to work on a fixed price basis if you're able and will prefer freelancers that can offer this. After submitting your proposal, we will send you the link to the images and you can bid from there. Also, do not use AI to respond to your questions - we want to hear from YOU!
